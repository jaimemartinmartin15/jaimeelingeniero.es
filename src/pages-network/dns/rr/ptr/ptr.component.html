El registro PTR es usado para resoluciones inversas. Dado una ip devuelve el nombre de dominio. El formato del registro es el siguiente: 

  <nombre-especial>  [<ttl>]  [<clase>]  PTR  <nombre>
nombre-especial
Realmente cuando otra máquina quiere saber el dominio asociado a una IP, no envía la IP como tal, sino que la convierte a un dominio especial llamado in-addr.arpa. Este dominio consiste en dar la vuelta a los números de la IP y concatenarle in-addr.arpa. Por ejemplo la IP 198.162.1.50 acabaría en 50.1.162.198.in-addr.arpa.
nombre
Es el nombre asociado a la IP por la que se pregunta. Debe ser un nombre oficial.

Pueden existir múltiples registros para direrentes IP y que devuelvan el mismo dominio.
