El registro MX contiene el nombre de dominio de una máquina que permite recibir correo electrónico. El formato del registro es el siguiente: 

  <dominio>  [<ttl>]  [<clase>]  MX  <preferencia>  <servidor>
dominio
Nombre de dominio del correo electrónio. Si por ejemplo vamos a enviar un correo a jaimemartinmartin15@gmail.com el valor sería gmail.com
preferencia
En caso de tener varias máquinas con servidores de correo activas, podemos especificar cual tiene preferencia. Cuanto más bajo sea el número, más preferencia tiene.
servidor
Es el nombre de dominio de la máquina que ejecuta el servidor de correo.

Se permite tener multiples registros de este tipo con la misma preferencia. En tal caso ambos tienen la misma prioridad.
