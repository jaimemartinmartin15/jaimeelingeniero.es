Los datos de este registro continen la dirección IPv4 de una máquina. 

  <host>  [<ttl>]  [<class>]  A  <address>
host
Es el nombre de la máquina (por ejemplo: www).
address
Es la dirección IP de la máquina en formato separado por puntos. Por ejemplo: 192.168.1.123

Si una máquina tiene varias direcciones IP, el servidor DNS debe tener un registro diferente para cada dirección IP.
