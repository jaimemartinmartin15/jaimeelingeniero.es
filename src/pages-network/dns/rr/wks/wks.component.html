El registro WKS significa Well Known Services (servicios bien conocidos). Permite enumerar los servicios conocidos que proporciona una máquina. El formato de los datos es el siguiente: 

  <máquina>  [<ttl>]  [<class>]  WKS  <dirección>  <protocolo>  <servicios>
máquina
Es el nombre de la máquina que proporciona los servicios.
dirección
Es la dirección IP de la máquina que proporciona los servicios.
protocolo
Es un número de protocolo IP. Por ejemplo 6 para TCP o 17 para UDP.
servicios
Son los servicios que están activos. Es un campo de bits de longitud variable. Si el bit es 1, el servicio esta activo. Si el bit es 0 o no está, el servicio no está activo. La posición del bit identifica el puerto. Por ejemplo, si el protocolo es TCP, el bit 26 identifica el puerto 25 (SMTP).

Actualmente por motivos de seguridad está desaconsejado usar este registro, y para comprobar si una máquina proporciona un servicio lo mejor es tratar de conectarse a él directamente.
