El registro CNAME es un alias para un nombre de dominio. El formato del registro es el que sigue: 

  <alias>  [<ttl>]  [<class>]  CNAME  <host>
alias
Es el nuevo nombre que queremos poner a la máquina.
host
Es el nombre que realmente tiene la máquina, el oficial. No puede ser directamente una IP.

Un ejemplo de uso de este registro sería si cambiamos el nombre de una máquina. En este caso podemos crear un registro de este tipo donde el alias sea el nombre viejo y el host el nombre nuevo. De esta forma los que sigan accediendo mediante el nombre antiguo son enviados al nuevo nombre.

Otro caso de uso sería para múltiples servicios corriendo en la misma máquina en diferentes dominios (que apuntan a la misma IP). Por ejemplo ftp.example.com y www.example.com. Si ambos servicios están en la misma máquina podemos crear registros CNAME que apunten al dominio base example.com, y en caso de cambiar la IP, solo tendremos que cambiar el registro A del dominio base.
